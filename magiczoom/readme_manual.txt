INSTALLATION:

IMPORTANT: If you use Magento Compiler, disable it. After installation is complete, go to Magento Compiler and run the compilation process.

1. Unzip the contents and upload the app, js and skin folders to your Magento directory.

2. Go to 'System -> Cache Management' and refresh the cache.

3. Configure the Magic Zoom module through the 'Magic Toolbox -> Magic Zoom settings' menu in the Magento admin panel.

4. You've now installed the demo version of Magic Zoom!

5. To upgrade, buy Magic Zoom and overwrite the magiczoom.js file with the same file from the full version.
